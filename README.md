Audiora is a premium music streaming and offline playback application for Android. Designed with a stunning glassmorphic UI, it provides a seamless and completely ad-free music experience by leveraging high-quality 320kbps streams.

## ✨ Features

- **Ad-Free Streaming:** Listen to your favorite tracks without any interruptions.
- **High-Quality Audio:** Fetches 320kbps MP4/M4A audio streams directly via JioSaavn.
- **Spotify Integration:** Import your entire Spotify playlist effortlessly with just a link.
- **Offline Downloads:** Download songs locally to listen when you're on the go without internet access.
- **Dynamic Synchronized Lyrics:** Sing along with live, perfectly synced lyrics.
- **Built-in Equalizer:** A native multi-band equalizer with presets to fine-tune your listening experience.
- **Smart Home Screen:** Features "Speed Dial" and "Quick Picks" that dynamically adapt to your recently played tracks and liked songs, styled beautifully like modern premium apps.
- **Beautiful UI:** Built fully in Jetpack Compose featuring smooth animations, dark modes, and dynamic palette extractions that theme the player based on the album art.

## 📥 Installation

1. Go to the [Releases](https://github.com/Astreas-Core/Audiora/releases) page.
2. Download the latest `Audiora-vX.X.X.apk`.
3. Open the file on your Android device.
4. If prompted, go to settings and allow **"Install from unknown sources"**.
5. Install and enjoy!

## 🛠️ Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Toolkit:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Dependency Injection:** [Koin](https://insert-koin.io/)
- **Media Playback:** [ExoPlayer (Media3)](https://developer.android.com/media/media3/exoplayer)
- **Networking:** [OkHttp3](https://square.github.io/okhttp/) & [Coroutines](https://kotlinlang.org/docs/coroutines-overview.html)
- **Image Loading:** [Coil](https://coil-kt.github.io/coil/)
